# whatsapp-api-nodejs
whatsapp API created using nodejs and whatsapp-web.js

### Setup:

clone the repo:

```bash
git clone https://github.com/darijadev/whatsapp-api-nodejs.git
```

cd into the folder

```bash
cd whatsapp-api-nodejs
```

install dependencies

```bash
npm install
```

run the server

```bash
npm start
```
